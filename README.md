# 🏛️ NEU Library Visitor Log System

A kiosk-style web application designed for **New Era University** to manage and track library attendance using RFID technology and institutional email authentication.

## Features

### Visitor Terminal (The Kiosk)
* **RFID Integration:** Supports quick tap entry using school ID cards.
* **Dual Authentication:** Alternative login via Institutional Google Email (`firsname.lastname@neu.edu.ph`).
* **Purpose Tracking:** Visitors select their purpose of visit (Reading, Thesis, PC Use, etc.).
* **Responsive Design:** Optimized for desktop monitors, tablets, and mobile devices.
* **Real-time Clock:** Displays current time for a professional kiosk experience.

### Admin Dashboard (The Management)
* **Statistics & Analytics:** Real-time visitor counts for **Daily, Weekly, and Monthly** periods.
* **Security Management:** Ability to **Block/Restrict** specific users from entering.
* **Patron Management:** Manage a dedicated Blocked List to restore access when needed.
* **PDF Reports:** Generate and download professional attendance reports using the FPDF library.

## Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Libraries:** FPDF (for PDF Generation)
